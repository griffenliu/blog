个人搭建hexo博客的过程

  1. 下载和安装git - http://git-scm.com/download/
  2. 下载和安装node - https://nodejs.org/en/
  3. 安装hexo - https://hexo.io/
    • npm install hexo-cli -g
    • cd
    • hexo init blog
    • cd blog
    • npm install
    • hexo server
    • 使用浏览器进入http://localhost:4000浏览站点
  4. 修改配置文件
  5. 安装主题 - hexo-theme-modernist

    • 修改配置文件_config.yml
    • 修改多语文件 - 模版下的languages文件夹

    • 设置好相应内容
    • 多说的域名地址去掉.duoshuo.com剩余的部分即为short-name
